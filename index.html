Welcome to Magdeburg Full Stack JavaScript

Let's build a community for JavaScript in and around Magdeburg.

Whether you're a beginner wanting to ask questions or a senior developer wanting to discuss web development, everybody is welcome.

We will have monthly talks around Angular, Node.js and related topics like microservices and Docker.

Apps built with React or Angular 2 are used by companies like Google and Facebook to build the latest generation of their web apps.

The SciFlow online editor has been built using Angular 2, RxJS and Node.js. We'll let you in on the pitfalls of working with such a new framework and give you hints on how to get started with your own app.
